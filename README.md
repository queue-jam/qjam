## Why QJAM Matters

No one likes closed source software that barely works.

## Core Features

- **Fully Open Source**: This app will always be open source, not for profit.
- **Accountless**: Users can create rooms and queue songs without authentication.
- **Multi-User Support**: Rooms support unlimited parties.

## Getting Started

### Prerequisites

Before installing, ensure you have the following:

- Nix

---

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/queue-jam/qjam.git
   cd qjam
   ```

2. **Install dependencies:**

   ```bash
   nix develop
   ```
